# Tello Programming(JavaScript)

1. PCをTelloのWi-Fiに接続（TelloのWi-Fiを調べておくこと）
2. cmd.exe (Windows)またはTerminal.app(Mac)で以下のコマンドを実行

```
node tello.js  // ←プログラムを書いたファイル
```

## Node.js Install

https://nodejs.org/ja/

## dgram Install

```
npm init
```

```
npm install dgram
```
